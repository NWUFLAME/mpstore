# 在js中使用

# 访问全局store

可以使用this.globalState.xxx和this.globalGetters.yyy访问全局state和getters。 注意：为了项目的可维护性和代码最佳实践，不要直接在页面中修改globalState，请使用如下修改全局Store的方式。

# 修改全局store

this.dispatchGlobal('xxx', payload)

this.commitGlobal('yyy', payload)

# 访问页面store

可以使用this.state.xxx和this.getters.yyy访问state和getters, 但是不论是state, 还是getters, 也可以一律从this.data中获取。 例如, 要获取state.xxx和getters.yyy, 只需要

const { xxx, yyy } = this.data;

注意：为了项目的可维护性和代码最佳实践，不要直接在页面中修改state，请使用如下修改页面store的方式。

# 修改页面store

this.dispatch('xxx', payload)

this.commit('yyy', payload)
